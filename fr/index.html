PentestBox

PentestBox est un Open Source préconfigurés Portable Penetration Testing Environment pour le système d'exploitation Windows

Pourquoi une autre distribution pentesting?

PentestBox ne ressemble à aucune autre distribution linux pentesting qui fonctionne soit dans une machine virtuelle ou sur un environnement à double amorçage.
Il fournit essentiellement tous les outils de sécurité comme un paquet de sécurité et vous permet de les exécuter en mode natif sur Windows. Ceci élimine efficacement l'exigence des machines virtuelles ou des environnements de DualBoot sur les fenêtres.
Il a été créé parce que plus de 50% des utilisateurs de distribution de tests de pénétration utiliser des machines virtuelles pour exécuter ces distributions sur le système d'exploitation Windows.[ Source]

Caractéristiques

insert_emoticon

Facile à utiliser

Il est un utilitaire de ligne de commande simple.

group

Design Simple.

Le vert traditionnel sur la borne noire est livré avec une touche moderne.

flash_on

Performance

PentestBox fonctionne directement sur des machines hôtes au lieu de compter sur des machines virtuelles, apportant des améliorations évidentes dans la performance.

build

Aucune Dépendances

Toutes les dépendances requises par les outils sont à l'intérieur PentestBox, donc vous pouvez même exécuter PentestBox sur les fenêtres fraîchement installés sans aucun tracas.

usb

Portable

PentestBox est entièrement portable, alors maintenant vous pouvez réaliser votre propre Test de Pénétration Environnement sur une clé USB. Il prendra soin de toutes les dépendances et la configuration requises pour exécuter des outils.

extension

Utilitaires Linux

PentestBox contient près de tous les services publics comme linux bash, chat, chmod, curl, git, gzip, ls, mv, ps, ssh, sh, uname et d'autres. Il contient même votre éditeur de texte favori "vim". Pour la liste complète, s'il vous plaît regardertools.pentestbox.org/#linux-utilities.

settings

Non Pilotes Problème

Windows a déjà un grand soutien de pilotes pour les cartes graphiques et les systèmes sans fil, donc maintenant vous avez pas à vous soucier des problèmes de compatibilité du pilote. Pour exemple- maintenant, vous pouvez utiliser la puissance de votre GPU pour casser hashes utilisant RainbowCrack du projet, qui est pas compatible sur un environnement comme Linux.

view_module

Modular

Dans la fabrication de PentestBox nous avons inclus seulement les meilleurs outils, mais dans ce processus, nous aurions pu manquer quelques outils que vous pourriez utiliser. Dans ce cas, vous pouvez facilement installer ces outils à l'aidetoolsmanagerprésent à l'intérieur PentestBox, il peut installer / mettre à jour les outils / Désinstaller qui ne sont pas déjà dans PentestBox.

memory

Moins Utilisation de la mémoire

PentestBox est exécuté sur la machine hôte sans avoir besoin d'une machine virtuelle. Ainsi, il n'a besoin de 20 Mo pour le lancement par rapport à au moins 2 Go de RAM nécessaire pour l'exécution de distributions de machines virtuelles.

storage

Moins Utilisation du disque

Il est très léger sur le disque ainsi, il faut moins d'un tiers de ce que linux distro pentesting fait.

computer

Navigateur Inbuilt

PentestBox contient une version du navigateur Mozilla Firefox avec presque tous ses addons de sécurité. Pour connaître la liste complète des addons, s'il vous plaît cliquer surIci.

update

Mise à jour Feature

Il contient une fonctionnalité de mise à jour à travers lequel vous pouvez garder vos outils mis à jour. Pour en savoir plus sur la fonction de mise à jour, cliquez s'il vous plaîtIci.

device_hub

Peut être partagé sur le réseau

Considérons un environnement où vous vouliez utiliser PentestBox sur de nombreux ordinateurs au bureau, laboratoire, etc. Au lieu d'installer PentestBox sur chaque ordinateur, il vous suffit de l'installer sur un ordinateur et partager ce dossier en tant que lecteur aux autres ordinateurs le même réseau. Pour en savoir plus sur cette fonctionnalité, cliquez surIci.

laptop_windows

système 32 bits supporté

La plupart des utilisateurs ne sont pas préoccupés par le soutien de 32 bits, car ils utilisent normalement les systèmes 64 bits. Mais cela a été introduit pour transformer les systèmes bas de gamme dans un environnement pentesting. Juste pour vous donner une idée, je l'ai testé sur PentestBoxCette $200 machine, Et il a Marché sans Problemme.

PentestBox

PentestBox est un Open Source préconfigurés Portable Test de Pénétration environnement pour le système d'exploitation Windows. It is based on ConEmu Et cmder. Crédits va à leurs développeurs pour fournir une telle plate-forme formidable pour construire PentestBox.
