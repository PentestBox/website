PentestBox

PentestBox es un ambiente para pruebas de penetración portable realizado para sistemas operativos Windows

Por qué otra distribución de Pentesting?

PentestBox no es como otra distribución linux para pentesting que corre en una máquina virtual o en un ambiente de doble inicio.
Escencialmente provee todas las herramientas de seguridad en una sola y te permite ejecutarlas natívamente en Windows. Esto elimina el requerimiento de máquinas virtuales o incios dobles en entornos Windows.
Fué creada por que mas del 50% de las personas que realizan Penetration Testing utilizan máquinas virtuales para correr distribuciones de seguridad en ambientes Windows.[ Fuente]

Features

insert_emoticon

Fácil de usar

Es una simple consola.

group

Diseño Simple

La tradicional consola verde en negro con un toque moderno.

flash_on

Rendimiento

PentestBox se ejecuta directamente en la máquina windows sin tener que correrlo en una máquina virtual, brindando gran poder en la performance.

build

No requiere dependencias

Todas las dependencias requeridas por las herramientas se encuentran dentro de PentesBox, permitiéndote correr PentestBox en un Windows recién instalado sin ningún problema.

usb

Portable

PentestBox es totalmente portable, ahora podrás llevar tu entorno de pruebas de penetración en un USB. El programa se encargará de todas las dependencias y configuraciones requerídas para funcionar.

extension

Utilidades de Linux

PentestBox contiene casi todas las utilidades de linux como bash, cat, chmod, curl, git, gzip, ls, mv, ps, ssh, sh, uname entre otras. También contiene tu editor de textos favorito "vim". Para una lista completa echa un vistazo a tools.pentestbox.org/#linux-utilities.

settings

Sin necesidad de drivers

Windows actualmente soporta gran cantidad de placas gráficas y sistemas de wifi, así que no deberás preocuparte por los problemas de compatibilidad con los drivers. Por ejemplo, ahora podrás usar el poder de tu GPU para crackear hashes usando RainbowCrack, el cual no es compatible en entornos Linux.

view_module

Modulár

In el proceso de creación de PentestBox solo incluímos las mejores herramientas, pero seguramente pudímos haber omitido algunas que quieras usar. En ese caso puedes fácilmente instalar esas herramientas usando "toolsmanager" que se encuentra de PentestBox, el cual puede instalar/actualizar y desinstalar aplicaciones que no están actualmente en PentestBox.

memory

Menos uso de memoria

PentestBox se ejecuta sin la necesidad de una máquina virtual. Solo necesita 20 MB para ejecutarse comparado con al menos los 2GB de RAM que se necesitan para ejecutar un entorno virtual.

storage

Menor uso de disco

Es muy liviando con el espacio, requiere menos de la tercera parte de lo que una distribució de Pentesting utiliza.

computer

Navegador Embedido

PentestBox contiene una version de Mozilla Firefox con casi todos los agregados de seguridad. Para conocer la lista completa de todos los agregados, por favor haz clic aquí.

update

Actualizaciones

Contiene una característica que permite actualizar las herramientas y así contar con sus versiones actualizadas. Para conocer mas sobre esta característica, por favor haz clic aquí.

device_hub

Puede ser compartido en una red

Imagina un entorno donde necesites usar PentestBox en varias computadoras de la oficina, laboratorio, etc. En vez de instalar PentestBox en cada computadora, puedes instalarlo en una y compartir la carpeta como una unidad a las otras computadoras en la misma red. Para conocer acerca de esta característica, haz clic aquí.

laptop_windows

Soporte 32 Bits

La mayoría de los usuarios no están preocupados acerca del soporte de 32 Bits debido a que normalmente utilizan sistemas de 64 bits. Pero esto permite transformar un sistema de 32 bits en un ambiente de PenTest. Para darte una idea, PentestBox fué probado en esta máquina de 200 dolares, y se ejecutó realmente fluido y sin problemas.

PentestBox

PentestBox es un ambiente para pruebas de penetración portable realizado para sistemas operativos Windows. Está basado en ConEmu y cmder. Los créditos van para sus desarrolladores que brindaron la excelente plataforma para construir PentestBox.
