PentestBox

O PentestBox é um ambiente de Pentest pré-configurado, portátil e opensource para o Windows.

Por que outra distribuição de Pentest?

O PentestBox não é como outra distribuição Linux qualquer que roda ou em uma máquina virtual ou em dual boot.
Ele traz todas as ferramentas de pentest em um pacote e te deixa rodá-las nativamente no Windows. Isso elimina a necessidade de máquinas virtuais ou de dual boot.
Ele foi criado pois mais de 50% dos usuários de distribuições para pentest usam máquinas virtuais para rodá-las no Windows.[ Fonte]

Características

insert_emoticon

Fácil de Usar

Uma simples ferramenta em linhas de comando.

group

Design Simples

O tradicional terminal preto e verde com um toque moderno.

flash_on

Performance

O PentestBox roda diretamente no seu computador ao invés de precisar de máquinas virtuais, trazendo ganhos em performance.

build

Nenhuma dependência necessária

Todas as dependências necessárias estão dentro do PentestBox, então você pode até rodar o PentestBox sem problemas em um Windows instalado recentemente.

usb

Portátil

O PentestBox é totalmente portátil, então agora você pode carregar seu ambiente de Pentest em um pen drive USB. Ele toma conta de todo o resto para você.

extension

Comandos Linux

O PentestBox contêm quase todos os comandos Linux essenciais como bash, cat, chmod, curl, git, gzip, ls, mv, ps, ssh, sh, uname e outros. Ele até vem com seu editor de texto favorito "vim". Para a lista completa, visite tools.pentestbox.org/#linux-utilities.

settings

Sem Problemas de Drivers

O Windows já tem um grande suporte para drivers de placas de vídeo and e sistemas wireless, você não precisa mais se preocupar com compatibilidade de drivers. Por exemplo, agora você pode usar o poder de sua GPU para crackear hashes usando o RainbowCrack, o que não é compativel em um sistema como o Linux.

view_module

Modular

Na criação do PentestBox nós incluimos somente as melhores ferramentas, mas nesse processo nós podemos ter esquecido algumas que você quer usar. Nesse caso, você pode facilmente instalar essas ferramentas usando o toolsmanager dentro do PentestBox, ele pode instalar/atualizar/desinstalar ferramentas que não estão presentes no PentestBox.

memory

Menos Uso de Memória

O PentestBox roda no máquina host sem a necessidade de uma máquina virtual. Ele só precisa de 20 MB para rodar, contra o mínimo de 2 GB de RAM necessárias para rodar uma máquina virtual.

storage

Menor uso de Disco

Ele também é muito leve, necessitando de apenas um terço do que uma distro de pentest precisa.

computer

Navegador Incluso

O PentestBox contém uma versão do Mozilla Firefox com quase todos os addons de segurança. Para ler a lista completa de addons, clique aqui.

update

Ferramenta de Atualização

Ele contém um sistema de atualização que mantém todas as suas ferramentas atualizadas. Para saber mais sobre esse sistema, clique aqui.

device_hub

Pode Ser Compartilhado Pela Rede

Imagine uma situação onde você quer usar o PentestBox em vários computadores de um escritório, laboratório, etc. Ao invés de instalar o PentestBox em cada computador, você pode simplesmente instalar em um computadror e compartilhar essa pasta com os outros em uma mesma rede interna. Para saber mais sobre essa função, clique aqui.

laptop_windows

Suporte à sistemas 32-bits

A maioria dos usuários não se preocupam com suporte à 32-bits porque normalmente eles usam sistemas 64-bits, mas essa característica foi adicionada para ajudar pentesters com sistemas low-end. Para você ter uma ideia, eu testei o PentestBox em uma máquina de USD$200, e ele rodou sem problemas.

Selecione seu download à direita

Existem dois tipos de instaladores: um com Metasploit e outro sem.
Programas anti-vírus e firewalls deverão ser desativados para instalar e usar a versão com Metasploit.
Baixe qualquer um dos tipos de instalador clicando no respectivo botão à direita.
Por padrão, o PentestBox é instalado em C:/PentestBox/, e para o funcionamento sem problemas, não mude o caminho.
Agora acesse tools.pentestbox.org e docs.pentestbox.org para aprender como usar as ferramentas.
Se você tiver qualquer problema ou pergunta, acesse faq.pentestbox.org ou poste seu problema forum.pentestbox.org. Se conecte com a gente no Facebook ou Twitter para receber atualizações sobre o PentestBox.
Achou interessante esse projeto? Existem várias maneiras de contribuir, acesse docs.pentestbox.org/contributing

PentestBox

O PentestBox é um ambiente de Pentest pré-configurado, portátil e opensource para o Windows. É baseado no ConEmu e o cmder. Créditos para os desenvolvedores por produzirem uma plataforma incrivel para criar o PentestBox.
